# Trade Tracker Dashboard

## Overview

Trade Tracker Dashboard is a personal portfolio analytics platform designed to help traders and investors track, analyze, and optimize their trading performance.

The application enables users to record open and closed trades, monitor profit and loss, calculate return on investment (ROI), track capital growth, analyze performance over multiple time horizons, and maintain a historical trading journal.

Built as a full-stack data-driven application using Python, SQLite, Pandas, Plotly, and Streamlit, the platform combines portfolio management, analytics, and visualization into a single dashboard.

---

## Key Features

### Trade Management

* Create and manage open and closed trades
* Record trade metadata, notes, and external references
* Update existing trades
* Close open trades at a later date
* Delete trades when needed

### Portfolio Analytics

* Real-time profit and loss calculations
* Return on Investment (ROI) tracking
* Current capital monitoring
* Open trade exposure tracking
* Win-rate analysis

### Interactive Dashboard

* Equity Curve (capital growth over time)
* Daily profit/loss analytics
* Weekly profit/loss analytics
* Monthly profit/loss analytics
* Individual trade performance analysis
* Custom date-range filtering

### Data Persistence

* SQLite database for structured storage
* Automatic Excel backup generation
* Local-first architecture with no cloud dependency

### User Configuration

* Profile management
* Starting capital tracking
* Multi-currency support
